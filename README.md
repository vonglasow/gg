gg for Git Graph
================

This little code take commit for a repository and create a graph with
graphviz.

Requirements
============

git version 1.7.9.5
dot - graphviz version 2.26.3
libpng
php

Usage
=====

```sh
./bin/gg.sh hoaproject/Core
```

Disclaimer
==========
This is a quick and dirty POC.
